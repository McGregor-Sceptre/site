## Deployment

This site is configured to be deployed via GitHub Pages:

1. Go to repository Settings
2. Navigate to Pages section
3. Set Source to "Deploy from a branch"
4. Select the main/master branch
5. The site will be available at hungros.com (CNAME configured)

## Domain Configuration

The CNAME file is already set up to point to `hungros.com`. Make sure your DNS settings point to GitHub Pages:

- Add an A record pointing to GitHub Pages IP addresses:
  - 185.199.108.153
  - 185.199.109.153
  - 185.199.110.153
  - 185.199.111.153
- Or add a CNAME record pointing to `mcgregor-sceptre.github.io`
